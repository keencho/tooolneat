진법 변환기란?

진법 변환기는 2진수, 8진수, 10진수, 16진수를 실시간으로 상호 변환하는 도구입니다. 프로그래밍, 컴퓨터 과학 학습, 디지털 시스템 이해에 유용합니다.

사용 방법

  1. 원하는 진법의 입력 필드에 숫자를 입력합니다
  2. 다른 진법의 값이 자동으로 변환되어 표시됩니다
  3. 어떤 필드에서든 입력하면 나머지가 동시에 변환됩니다

진법 종류

  • 2진수 (Binary): 0과 1만 사용, 컴퓨터의 기본 연산 단위
  • 8진수 (Octal): 0-7 사용, Unix 파일 권한 표기에 사용
  • 10진수 (Decimal): 일상에서 사용하는 숫자 체계
  • 16진수 (Hexadecimal): 0-9, A-F 사용, 색상 코드, 메모리 주소 등

자주 묻는 질문

왜 2진수를 사용하나요?

컴퓨터는 전기 신호의 켜짐(1)과 꺼짐(0) 두 가지 상태로 동작하기 때문에 2진수를 기본으로 사용합니다.

16진수는 어디에 사용되나요?

웹 색상 코드(#FF0000), 메모리 주소, MAC 주소 등 IT 분야에서 널리 사용됩니다.

음수도 변환할 수 있나요?

현재는 양의 정수만 지원합니다. 음수의 경우 2의 보수 표현이 필요합니다.
