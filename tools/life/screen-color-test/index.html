화면 색상 테스트

다양한 색상과 패턴으로 모니터의 색상 균일성, 색재현력, 밴딩 현상을 테스트합니다. 아래 버튼을 클릭하면 전체화면으로 전환됩니다.

단색

패턴

사용 방법

  • • 색상/패턴 버튼 클릭 → 전체화면 모드로 전환
  • • ← → 또는 클릭으로 색상/패턴 변경
  • • ESC 키로 종료
  • • 화면 전체를 천천히 살펴보며 색상 불균일이나 밴딩 현상이 있는지 확인하세요.

색상 균일성 테스트란?

색상 균일성(Color Uniformity): 화면 전체에서 동일한 색상이 균일하게 표시되는지 확인합니다. 모서리나 가장자리에서 색상이 다르게 보이면 균일성 문제가 있을 수 있습니다.

밴딩(Banding): 그라디언트에서 부드러운 전환 대신 띠 모양의 줄무늬가 보이는 현상입니다. 모니터의 색심도나 패널 품질과 관련이 있습니다.

색재현력(Color Gamut): 모니터가 얼마나 넓은 범위의 색상을 정확하게 표현할 수 있는지를 나타냅니다. 선명한 빨강, 초록, 파랑이 잘 표시되는지 확인하세요.

백라이트 블리딩(Backlight Bleed): 검정 화면에서 가장자리나 모서리에서 빛이 새어 나오는 현상입니다. 어두운 환경에서 확인하세요.

클릭 또는 ← → 키로 패턴 변경 | ESC로 종료
