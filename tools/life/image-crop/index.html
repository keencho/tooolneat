이미지 자르기

이미지를 원하는 영역으로 자릅니다. 다양한 비율 옵션을 지원합니다.

이미지를 드래그하거나 클릭해서 선택하세요

JPG, PNG, WebP, GIF 지원

원본: - 선택 영역: -
100% 브라우저 처리 - 이미지가 서버로 전송되지 않습니다

이미지 자르기 안내

이미지의 원하는 부분만 선택하여 자를 수 있습니다. SNS 프로필 사진, 썸네일 제작, 불필요한 부분 제거에 유용합니다.

사용 방법

  1. 이미지를 드래그하거나 클릭하여 업로드합니다
  2. 원하는 비율을 선택합니다 (자유, 1:1, 16:9 등)
  3. 자르기 영역을 드래그하여 위치를 조정합니다
  4. 모서리나 가장자리를 드래그하여 크기를 조정합니다
  5. '자르기 및 다운로드' 버튼을 클릭합니다

비율 안내

  • 1:1: 인스타그램, 프로필 사진
  • 4:3: 일반 사진, 프레젠테이션
  • 16:9: 유튜브 썸네일, 와이드스크린
  • 9:16: 인스타그램/틱톡 스토리, 릴스
