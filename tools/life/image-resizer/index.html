이미지 리사이즈란?

이미지 리사이즈는 이미지의 크기(해상도)를 변경하는 것입니다. 웹사이트 업로드, SNS 게시, 썸네일 생성 등 다양한 용도로 활용됩니다.

사용 방법

  1. 이미지를 드래그하거나 클릭하여 업로드합니다
  2. 픽셀 또는 퍼센트로 원하는 크기를 입력합니다
  3. '적용' 버튼을 클릭하여 리사이즈합니다
  4. '다운로드' 버튼으로 결과를 저장합니다

프리셋 안내

  • 1920×1080 (FHD): Full HD 해상도, 유튜브 썸네일
  • 1080×1080 (IG): 인스타그램 정사각형 포스트
  • 1200×630 (OG): Open Graph 이미지 (SNS 공유용)
  • 256×256, 128×128: 아이콘, 프로필 이미지

모든 처리는 브라우저에서 이루어지며 이미지가 서버로 전송되지 않습니다.
