음성을 텍스트로

마이크로 말하면 실시간으로 텍스트로 변환됩니다. 회의록 작성, 받아쓰기에 유용합니다.

브라우저 호환성 안내

이 기능은 Chrome, Edge, Safari에서 가장 잘 작동합니다. Firefox는 지원이 제한적입니다.

마이크 버튼을 눌러 시작하세요

텍스트를 직접 수정할 수 있습니다

브라우저 기반 처리 - 음성이 서버로 전송되지 않습니다

음성 인식 안내

Web Speech API를 사용하여 음성을 실시간으로 텍스트로 변환합니다. 회의록 작성, 인터뷰 기록, 받아쓰기 등에 유용합니다.

사용 방법

  1. 인식할 언어를 선택합니다
  2. 마이크 버튼을 클릭합니다
  3. 브라우저에서 마이크 권한을 허용합니다
  4. 말하면 실시간으로 텍스트가 표시됩니다
  5. 다시 클릭하면 인식이 중지됩니다

자주 묻는 질문

Q: 어떤 브라우저를 사용해야 하나요?

Chrome, Edge, Safari에서 가장 잘 작동합니다. Firefox는 Web Speech API 지원이 제한적입니다.

Q: 인식 정확도가 낮아요

조용한 환경에서 마이크에 가까이 대고 또박또박 말하면 정확도가 높아집니다. 외부 마이크 사용도 도움됩니다.

Q: 음성 데이터가 서버로 전송되나요?

Web Speech API는 브라우저에 내장된 기능을 사용합니다. 브라우저에 따라 클라우드 서비스를 사용할 수 있으나, 이 사이트로는 데이터가 전송되지 않습니다.

Q: 오프라인에서도 작동하나요?

대부분의 브라우저는 온라인 연결이 필요합니다. 일부 기기에서는 오프라인 음성 인식이 지원됩니다.
