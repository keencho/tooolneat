색상 피커 (스포이드)

화면 어디서나 색상을 추출합니다. 브라우저 바깥의 다른 앱(Slack, Figma 등)에서도 색상을 추출할 수 있습니다.

브라우저 미지원

이 기능은 Chrome 95+ 또는 Edge 95+ 브라우저에서만 사용할 수 있습니다. Safari, Firefox는 아직 지원하지 않습니다.

사용 팁

  • 브라우저 바깥의 앱에서도 색상을 추출할 수 있습니다
  • ESC 키를 누르면 취소됩니다

색상 피커(스포이드)란?

색상 피커(스포이드)는 화면의 어떤 위치에서든 색상을 추출할 수 있는 도구입니다. EyeDropper API를 사용하여 브라우저뿐만 아니라 다른 앱(Slack, Figma, Photoshop 등)에서도 색상을 추출할 수 있습니다.

사용 방법

  1. "색상 선택" 버튼을 클릭합니다
  2. 화면 어디서든 원하는 색상 위에 커서를 놓습니다
  3. 클릭하면 색상이 추출됩니다
  4. HEX, RGB, HSL 형식으로 복사할 수 있습니다

자주 묻는 질문

Q: 어떤 브라우저에서 사용할 수 있나요?

Chrome 95 이상, Edge 95 이상에서 사용 가능합니다. Safari와 Firefox는 아직 EyeDropper API를 지원하지 않습니다.

Q: 브라우저 바깥에서도 색상을 추출할 수 있나요?

네, 가능합니다. 스포이드가 활성화되면 화면 전체에서 색상을 추출할 수 있습니다. 다른 앱의 색상도 추출할 수 있습니다.

Q: 정확한 색상이 추출되지 않는 것 같아요

디스플레이 색상 프로필, 브라우저 렌더링 등에 따라 약간의 차이가 있을 수 있습니다. 정확한 색상이 필요하면 원본 소스의 색상 코드를 확인하세요.

모든 처리는 브라우저에서 이루어지며 서버로 전송되지 않습니다.
