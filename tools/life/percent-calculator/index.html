퍼센트 계산 안내

  • X의 Y% = X x (Y / 100)
  • X가 Y의 몇 %인지 = (X / Y) x 100
  • 변화율 = ((새값 - 이전값) / 이전값) x 100
  • 할인 가격 = 원가 x (1 - 할인율 / 100)

퍼센트 계산기란?

퍼센트 계산기는 다양한 백분율 계산을 쉽고 빠르게 수행할 수 있는 도구입니다. 일상생활에서 자주 사용되는 퍼센트 계산을 5가지 모드로 제공하여, 할인 계산, 비율 계산, 증감률 계산 등을 간편하게 할 수 있습니다.

주요 기능

  1. X의 Y% - 특정 값의 백분율을 계산합니다 (예: 200의 15%는?)
  2. X는 Y의 몇 %? - 두 값의 비율을 백분율로 표시합니다 (예: 30은 200의 몇 %?)
  3. 증감률 - 이전 값과 새로운 값 사이의 변화율을 계산합니다
  4. 증감 적용 - 특정 값에 백분율을 증가 또는 감소시킵니다
  5. 할인 계산 - 원래 가격에서 할인율을 적용한 최종 가격을 계산합니다

자주 묻는 질문

Q: 퍼센트(%)는 무엇인가요?

퍼센트는 '100당'이라는 의미로, 전체를 100으로 보았을 때 차지하는 비율을 나타냅니다. 50%는 전체의 절반(1/2), 25%는 1/4을 의미합니다.

Q: 증가율과 감소율은 어떻게 다른가요?

증가율은 양수(+)로, 감소율은 음수(-)로 표시됩니다. 예를 들어 100에서 120으로 변하면 +20% 증가, 100에서 80으로 변하면 -20% 감소입니다.

Q: 할인 계산은 어떻게 하나요?

원래 가격에서 할인율을 빼면 됩니다. 예를 들어 10,000원에서 30% 할인하면, 10,000 x 0.3 = 3,000원이 할인되어 최종 가격은 7,000원입니다.
