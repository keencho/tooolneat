이미지 회전/뒤집기

이미지를 90도, 180도 회전하거나 좌우/상하로 뒤집습니다.

이미지를 드래그하거나 클릭해서 선택하세요

JPG, PNG, WebP, GIF 지원

원본: - 결과: -
100% 브라우저 처리 - 이미지가 서버로 전송되지 않습니다

이미지 회전/뒤집기 안내

스마트폰에서 잘못된 방향으로 찍힌 사진을 바로잡거나, 거울 이미지를 만들 수 있습니다.

사용 방법

  1. 이미지를 드래그하거나 클릭하여 업로드합니다
  2. 원하는 회전/뒤집기 버튼을 클릭합니다
  3. 여러 작업을 순서대로 적용할 수 있습니다
  4. '다운로드' 버튼을 클릭하여 저장합니다

기능 설명

  • 왼쪽 90°: 시계 반대 방향으로 90도 회전
  • 오른쪽 90°: 시계 방향으로 90도 회전
  • 180°: 이미지를 완전히 뒤집기
  • 좌우 뒤집기: 거울처럼 좌우 반전
  • 상하 뒤집기: 위아래 반전
