ASCII/유니코드 변환기

문자를 ASCII, 유니코드 코드 포인트로 변환하거나, 코드를 문자로 변환합니다.

10진수(공백구분), 16진수(0x), 유니코드(\u), HTML(&#;) 형식 지원

ASCII 테이블 (32-126)

문자 DEC HEX 문자 DEC HEX 문자 DEC HEX

ASCII/유니코드 변환기란?

ASCII/유니코드 변환기는 문자를 코드 포인트로, 또는 코드를 문자로 변환하는 도구입니다. 프로그래밍, 웹 개발, 문자 인코딩 이해에 유용합니다.

사용 방법

  1. 문자 → 코드: 텍스트를 입력하면 10진수, 16진수, 유니코드 이스케이프, HTML 엔티티로 변환됩니다
  2. 코드 → 문자: 코드를 입력하면 해당 문자로 변환됩니다
  3. 복사 버튼으로 결과를 클립보드에 복사합니다

지원 형식

  • 10진수 (DEC): 공백으로 구분된 숫자 (예: 72 101 108 108 111)
  • 16진수 (HEX): 0x 접두사 (예: 0x48 0x65 0x6C)
  • 유니코드 이스케이프: \u 형식 (예: \u0048\u0065)
  • HTML 엔티티: &# 형식 (예: He)

자주 묻는 질문

ASCII와 유니코드의 차이점은?

ASCII는 영문 기반 128개 문자만 지원하고, 유니코드는 전 세계 모든 문자(한글, 이모지 등)를 지원합니다.

\u 형식은 어디에 사용하나요?

JavaScript, Python, Java 등 프로그래밍 언어에서 문자열에 특수 문자를 표현할 때 사용합니다.

HTML 엔티티는 언제 필요한가요?

HTML에서 특수 문자(<, >, & 등)를 안전하게 표시하거나, 키보드로 입력하기 어려운 문자를 사용할 때 필요합니다.
