색상 형식 안내

  • • HEX: 웹에서 가장 많이 사용되는 16진수 형식 (#RRGGBB)
  • • RGB: 빨강, 초록, 파랑 값 (0-255)
  • • HSL: 색상(Hue), 채도(Saturation), 명도(Lightness)

색상 변환기란?

색상 변환기는 웹 개발과 디자인에서 사용하는 다양한 색상 형식(HEX, RGB, HSL)을 상호 변환하는 도구입니다. CSS, 그래픽 디자인, 브랜딩 작업에서 색상 코드를 다룰 때 필수적입니다. 또한 보색, 밝기 조절, 채도 조절 등의 관련 색상도 함께 확인할 수 있어 색상 팔레트 구성에 유용합니다.

사용 방법

  1. 색상 선택기를 클릭하여 원하는 색상을 선택하거나, HEX 코드를 직접 입력합니다.
  2. 빠른 선택 팔레트에서 자주 사용하는 색상을 클릭할 수 있습니다.
  3. RGB 슬라이더로 빨강, 초록, 파랑 값을 개별적으로 조절합니다.
  4. 투명도(Alpha) 슬라이더로 불투명도를 조절합니다.
  5. 각 형식(HEX, RGB, HSL, RGBA, HSLA, HEX8) 옆의 복사 버튼으로 원하는 형식을 복사합니다.
  6. 관련 색상 섹션에서 보색, 밝게/어둡게, 채도 높은 색상을 확인하고 클릭하여 적용합니다.

자주 묻는 질문

Q: HEX, RGB, HSL 중 어떤 형식을 사용해야 하나요?

HEX는 CSS에서 가장 간결하고 널리 사용됩니다. RGB는 프로그래밍에서 색상 계산 시 편리합니다. HSL은 색상(Hue), 채도(Saturation), 명도(Lightness)로 직관적인 색상 조절이 필요할 때 유용합니다.

Q: HEX8은 무엇인가요?

HEX8은 일반 HEX 코드(#RRGGBB)에 투명도 값 2자리(AA)를 추가한 형식(#RRGGBBAA)입니다. 최신 브라우저에서 지원되며, RGBA 대신 더 간결하게 투명도를 표현할 수 있습니다.

Q: 보색(Complementary Color)은 무엇인가요?

보색은 색상환에서 정반대(180도) 위치에 있는 색상입니다. 서로 대비가 강해 시각적 강조에 효과적이며, 디자인에서 악센트 색상으로 자주 활용됩니다.
