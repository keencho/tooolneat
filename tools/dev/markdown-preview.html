마크다운 미리보기

마크다운을 작성하고 실시간으로 미리보기 하세요. 헤더, 목록, 코드 블록, 테이블 등 다양한 마크다운 문법을 지원합니다.

미리보기 테마:
지원: GFM 코드 하이라이트 수식 (MathJax) 테이블 체크박스
마크다운 입력
미리보기
마크다운을 입력하면 여기에 미리보기가 표시됩니다.

마크다운이란?

마크다운(Markdown)은 텍스트를 HTML로 변환하는 간단한 마크업 언어입니다. 개발자 문서, README 파일, 블로그 포스트 작성에 널리 사용됩니다. 간단한 문법으로 제목, 목록, 링크, 이미지, 코드 블록 등을 쉽게 작성할 수 있습니다.

지원하는 문법

  • 제목: # H1, ## H2, ### H3 등
  • 강조: **굵게**, *기울임*, ~~취소선~~
  • 목록: - 또는 * 로 순서 없는 목록, 1. 2. 3. 으로 순서 있는 목록
  • 체크박스: - [ ] 미완료, - [x] 완료 (Task List)
  • 링크: [링크 텍스트](URL)
  • 이미지: ![대체 텍스트](이미지URL)
  • 코드: `인라인 코드` 또는 ```언어 코드블록``` (구문 강조 지원)
  • 수식: $인라인 수식$ 또는 $$블록 수식$$ (MathJax/LaTeX)
  • 인용: > 인용문
  • 테이블: | 헤더1 | 헤더2 | 형식의 GFM 테이블
  • 수평선: --- 또는 ***

자주 묻는 질문

Q: 마크다운은 어디서 사용되나요?

GitHub, GitLab, Notion, Slack, Discord 등 많은 플랫폼에서 마크다운을 지원합니다. README 파일, 기술 문서, 블로그 포스트 작성에 표준으로 사용됩니다.

Q: 입력한 내용이 서버로 전송되나요?

아니요, 모든 변환은 브라우저에서 로컬로 처리됩니다. 입력한 내용은 외부로 전송되지 않으며 안전하게 사용할 수 있습니다.

Q: HTML로 내보낸 결과물을 어떻게 사용하나요?

'HTML 다운로드' 버튼으로 완전한 HTML 파일을 받을 수 있습니다. 이 파일을 웹서버에 업로드하거나 로컬에서 열어 확인할 수 있습니다.
