메타 태그 안내

메타 태그는 검색 엔진과 소셜 미디어에서 페이지가 어떻게 표시되는지 결정합니다. 올바른 메타 태그 설정은 SEO와 소셜 공유에 필수적입니다.

Open Graph

Facebook, LinkedIn 등에서 링크 공유 시 표시되는 정보를 정의합니다. og:title, og:description, og:image 등이 포함됩니다.

Twitter Card

Twitter에서 링크 공유 시 카드 형태로 표시됩니다. summary_large_image 타입을 사용하면 큰 이미지가 표시됩니다.

권장 사항

  • 제목: 50-60자 (검색 결과에서 잘리지 않음)
  • 설명: 120-160자 (핵심 내용을 간결하게)
  • 이미지: 1200x630px (Open Graph 최적)
  • 각 페이지마다 고유한 메타 태그 사용
