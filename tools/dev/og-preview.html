Open Graph란?

Open Graph는 Facebook이 만든 프로토콜로, 웹페이지가 SNS에서 공유될 때 제목, 설명, 이미지 등을 정의합니다. 대부분의 소셜 플랫폼이 이 프로토콜을 지원합니다.

필수 태그

  • og:title: 콘텐츠 제목 (60자 권장)
  • og:description: 콘텐츠 설명 (155자 권장)
  • og:image: 미리보기 이미지 URL
  • og:url: 페이지의 표준 URL

이미지 권장사항

  • 최소 크기: 1200 x 630 픽셀
  • 비율: 1.91:1
  • 파일 크기: 8MB 미만
  • 형식: JPG, PNG, GIF
