색상 대비 검사기

WCAG 웹 접근성 기준에 따라 색상 대비율을 검사합니다.

대비율

21:1

일반 텍스트

AA (4.5:1) 통과
AAA (7:1) 통과

큰 텍스트 (18pt+)

AA (3:1) 통과
AAA (4.5:1) 통과

추천 색상 조합

색상 대비 검사란?

웹 콘텐츠 접근성 지침(WCAG)에서 정의한 색상 대비율을 검사하여 시각 장애인이나 색각 이상자도 콘텐츠를 읽을 수 있는지 확인합니다.

WCAG 등급 기준

  • AA 일반 텍스트: 최소 4.5:1 대비율 필요
  • AA 큰 텍스트: 최소 3:1 대비율 필요 (18pt 이상 또는 14pt 굵게)
  • AAA 일반 텍스트: 최소 7:1 대비율 필요
  • AAA 큰 텍스트: 최소 4.5:1 대비율 필요

자주 묻는 질문

Q: 대비율은 어떻게 계산되나요?

두 색상의 상대 휘도를 계산한 뒤, (밝은색 + 0.05) / (어두운색 + 0.05) 공식으로 계산합니다.

Q: AA와 AAA의 차이는?

AA는 최소 기준, AAA는 권장 기준입니다. 법적 요구사항은 대부분 AA 등급입니다.

Q: 큰 텍스트의 기준은?

18pt(24px) 이상 또는 14pt(약 18.5px) 굵은 글씨를 큰 텍스트로 분류합니다.
