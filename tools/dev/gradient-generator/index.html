CSS 그라데이션 생성기

CSS 그라데이션을 시각적으로 만들고 코드를 복사하세요.

CSS 그라데이션이란?

두 개 이상의 색상이 부드럽게 전환되는 배경 효과입니다. 웹사이트에 세련된 디자인을 더할 수 있습니다.

사용 방법

  1. Linear(직선) 또는 Radial(원형) 타입 선택
  2. 색상을 추가하고 조절합니다
  3. 방향이나 각도를 설정합니다
  4. CSS 코드를 복사하여 사용합니다

자주 묻는 질문

Q: 모든 브라우저에서 작동하나요?

최신 브라우저 모두 지원합니다. IE는 지원하지 않습니다.

Q: 색상을 몇 개까지 추가할 수 있나요?

제한 없이 추가 가능합니다. 보통 2~4개가 자연스럽습니다.

Q: 텍스트에도 적용 가능한가요?

네, background-clip: text와 함께 사용하면 텍스트에도 적용됩니다.
