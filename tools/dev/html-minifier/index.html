HTML 압축이란?

HTML 압축(Minification)은 HTML 코드의 기능을 유지하면서 파일 크기를 줄이는 과정입니다. 불필요한 공백, 줄바꿈, 주석을 제거하여 웹 페이지 로딩 속도를 향상시킵니다.

압축 효과

  • 파일 크기 감소 (보통 10~30%)
  • 페이지 로딩 시간 단축
  • 서버 대역폭 절약
  • SEO 점수 향상

주의사항

  • pre, code, textarea 태그 내 공백은 유지됩니다
  • 압축된 코드는 가독성이 떨어집니다
  • 개발 시에는 원본을 보관하세요
