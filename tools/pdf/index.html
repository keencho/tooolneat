자주 묻는 질문

PDF 파일이 서버로 전송되나요?

아니요. 모든 PDF 처리는 브라우저 내에서 이루어집니다. 파일이 서버로 업로드되지 않아 개인정보가 안전합니다.

최대 파일 크기 제한이 있나요?

브라우저 메모리에 따라 다르지만, 일반적으로 50MB 이하의 PDF를 권장합니다. 대용량 파일은 처리 속도가 느려질 수 있습니다.

암호로 보호된 PDF도 처리할 수 있나요?

암호가 걸린 PDF는 먼저 암호를 해제해야 합니다. 현재 암호 해제 기능은 지원하지 않습니다.

어떤 브라우저를 사용해야 하나요?

Chrome, Firefox, Edge 등 최신 브라우저에서 사용할 수 있습니다. 최상의 경험을 위해 Chrome을 권장합니다.
