PDF 문서에 텍스트 워터마크를 추가하여 저작권을 표시하거나 문서 상태를 명시합니다.

사용 방법

  1. PDF 파일을 업로드합니다
  2. 워터마크 텍스트를 입력합니다
  3. 위치, 크기, 회전, 투명도, 색상을 조절합니다
  4. '다운로드' 버튼을 클릭합니다

활용 예시

  • 기밀 문서: "CONFIDENTIAL", "대외비"
  • 초안 문서: "DRAFT", "검토용"
  • 저작권 표시: 회사명, 작성자 이름
  • 샘플 문서: "SAMPLE", "견본"
