PDF 분할이란?

PDF 분할은 하나의 PDF 파일에서 원하는 페이지만 추출하거나 각 페이지를 개별 파일로 나누는 것입니다. 긴 문서에서 필요한 부분만 추출하거나, 스캔한 문서를 개별 파일로 분리할 때 유용합니다.

사용 방법

  1. PDF 파일을 드래그하거나 클릭하여 업로드합니다
  2. 추출할 페이지를 클릭하여 선택합니다 (여러 페이지 선택 가능)
  3. '선택 페이지 추출' 버튼을 클릭합니다
  4. 선택한 페이지만 포함된 새 PDF가 다운로드됩니다

자주 묻는 질문

Q: 여러 페이지를 한 번에 선택할 수 있나요?

네, 원하는 페이지를 여러 개 클릭하여 선택할 수 있습니다. 선택된 페이지는 파란색 테두리로 표시됩니다.

Q: 분할 후 페이지 순서를 변경할 수 있나요?

현재는 원본 순서대로 추출됩니다. 순서 변경이 필요하면 분할 후 PDF 병합 도구를 사용하여 재배열할 수 있습니다.

Q: 원본 PDF 파일이 변경되나요?

아니요, 원본 파일은 변경되지 않습니다. 선택한 페이지로 새로운 PDF 파일이 생성됩니다.

Q: 분할한 PDF의 품질이 저하되나요?

아니요, 원본의 품질이 그대로 유지됩니다. 텍스트, 이미지, 폰트 등 모든 요소가 동일하게 보존됩니다.

모든 처리는 브라우저에서 이루어지며 파일이 서버로 전송되지 않습니다.
