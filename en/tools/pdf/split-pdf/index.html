What is PDF Split?

PDF split extracts specific pages from a PDF or separates each page into individual files. Great for extracting needed sections from long documents or splitting scanned documents.

How to Use

  1. Drag or click to upload a PDF file
  2. Click pages to select them (multiple selection supported)
  3. Click 'Extract Selected Pages' button
  4. A new PDF with selected pages will download

Frequently Asked Questions

Q: Can I select multiple pages at once?

Yes, click multiple pages to select them. Selected pages are highlighted with a blue border.

Q: Can I change the page order after splitting?

Currently pages are extracted in original order. To reorder, use the PDF Merge tool after splitting.

Q: Is the original PDF file modified?

No, the original file remains unchanged. A new PDF is created with your selected pages.

Q: Is quality lost when splitting?

No, original quality is preserved. Text, images, fonts, and all elements remain identical.

All processing is done in your browser. No data is uploaded to servers.
