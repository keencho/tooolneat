About Meta Tags

Meta tags determine how your page appears in search engines and social media. Proper meta tag configuration is essential for SEO and social sharing.

Open Graph

Defines how links appear when shared on Facebook, LinkedIn, etc. Includes og:title, og:description, og:image and more.

Twitter Card

Controls how links appear as cards on Twitter. Using summary_large_image displays a large image preview.

Best Practices

  • Title: 50-60 characters (won't be truncated in search results)
  • Description: 120-160 characters (concise summary)
  • Image: 1200x630px (optimal for Open Graph)
  • Use unique meta tags for each page
