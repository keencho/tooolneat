What is URL Encoder/Decoder?

URL encoding (also known as percent-encoding) is a method for encoding special characters in URLs so they can be safely transmitted over the internet. Since URLs can only contain a limited set of ASCII characters, special characters like spaces, ampersands, and non-English characters must be converted to a format that can be universally recognized by web browsers and servers. This tool helps developers encode and decode URLs quickly and accurately.

How to Use

  1. Choose the operation mode by clicking "Encode" or "Decode" at the top of the tool.
  2. Select the encoding method: Component (for query parameters), URI (for full URLs), or Full (encode all special characters).
  3. Enter your URL or text string in the input field on the left.
  4. With "Live" mode enabled, the result updates automatically. Otherwise, click the Convert button.
  5. Use the "Copy" button to copy the result, or the swap button to exchange input and output.

Frequently Asked Questions

Q: What is the difference between encodeURIComponent and encodeURI?

encodeURIComponent encodes all special characters including /, ?, #, and &, making it ideal for encoding query parameter values. encodeURI preserves these URL-reserved characters, so it should be used when encoding a complete URL where you want to keep the structure intact. Use encodeURIComponent for values and encodeURI for full URLs.

Q: Why do I need to encode URLs?

URL encoding is necessary because URLs can only contain a specific set of characters from the ASCII character set. Characters like spaces, special symbols, and non-ASCII characters (like Chinese or Korean) must be encoded to be properly interpreted by web servers. Without encoding, these characters could break the URL or be misinterpreted.

Q: What does %20 mean in a URL?

%20 is the URL-encoded representation of a space character. When you see %XX in a URL, the XX represents a hexadecimal value of the character's ASCII code. For example, %20 is hex 20, which equals decimal 32, the ASCII code for a space. Similarly, %2F represents a forward slash (/) and %26 represents an ampersand (&).
