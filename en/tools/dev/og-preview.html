What is Open Graph?

Open Graph is a protocol created by Facebook that defines how web pages appear when shared on social networks. It specifies the title, description, image, and other metadata. Most social platforms support this protocol.

Essential Tags

  • og:title: Content title (60 chars recommended)
  • og:description: Content description (155 chars recommended)
  • og:image: Preview image URL
  • og:url: Canonical page URL

Image Recommendations

  • Minimum size: 1200 x 630 pixels
  • Aspect ratio: 1.91:1
  • File size: Under 8MB
  • Format: JPG, PNG, GIF
