What is Lorem Ipsum?

Lorem Ipsum is the standard dummy text used in the publishing and graphic design industries. It has been used since the 1500s and is widely used for layout and font testing.

It is used to preview designs before the actual content is ready. Since it is unreadable text, it helps readers focus on the layout rather than the content.
