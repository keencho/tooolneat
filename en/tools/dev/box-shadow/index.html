What is CSS Box Shadow?

box-shadow is a CSS property that adds shadow effects to elements. Multiple shadows can be layered by separating them with commas.

Syntax

box-shadow: offset-x offset-y blur spread color [inset];

Properties

  • offset-x: Horizontal offset (positive = right)
  • offset-y: Vertical offset (positive = down)
  • blur: Blur radius (larger = softer)
  • spread: Shadow size expansion/contraction
  • color: Shadow color
  • inset: Changes to inner shadow
