Robots.txt Generator

Generate robots.txt files to control search engine crawlers.

robots.txt Preview

Usage: Save the generated file as robots.txt in your website's root directory.

What is robots.txt?

robots.txt is a text file that tells web crawlers which parts of your site they can access. It's essential for search engine optimization (SEO).

Directives

  • User-agent: Specifies which crawler the rules apply to
  • Disallow: Paths to block from crawling
  • Allow: Paths to allow crawling
  • Sitemap: Location of your sitemap file
  • Crawl-delay: Wait time between requests

Notes

  • robots.txt is advisory, not enforced
  • Protect sensitive data by other means
  • File must be in the root directory
