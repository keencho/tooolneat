What is Markdown?

Markdown is a lightweight markup language that converts plain text into HTML. It's widely used for developer documentation, README files, and blog posts. With simple syntax, you can easily create headings, lists, links, images, code blocks, and more.

Supported Syntax

  • Headings: # H1, ## H2, ### H3, etc.
  • Emphasis: **bold**, *italic*, ~~strikethrough~~
  • Lists: - or * for unordered, 1. 2. 3. for ordered
  • Checkboxes: - [ ] incomplete, - [x] complete (Task List)
  • Links: [link text](URL)
  • Images: ![alt text](image URL)
  • Code: `inline code` or ```language code block``` (with syntax highlighting)
  • Math: $inline math$ or $$block math$$ (MathJax/LaTeX)
  • Blockquotes: > quoted text
  • Tables: | Header1 | Header2 | GFM table format
  • Horizontal rules: --- or ***

Frequently Asked Questions

Q: Where is Markdown used?

Many platforms support Markdown including GitHub, GitLab, Notion, Slack, and Discord. It's the standard format for README files, technical documentation, and blog posts.

Q: Is my content sent to a server?

No, all conversion happens locally in your browser. Your content is never transmitted externally and can be used safely with sensitive information.

Q: How do I use the exported HTML?

Use the 'Download HTML' button to get a complete HTML file. You can upload this file to a web server or open it locally in your browser.
