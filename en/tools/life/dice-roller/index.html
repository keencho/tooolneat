Dice Types Guide

D4: Tetrahedron (1-4) D6: Cube (1-6) D8: Octahedron (1-8) D10: Pentagonal (1-10) D12: Dodecahedron (1-12) D20: Icosahedron (1-20)

What is a Dice Roller?

A dice roller is a tool that lets you roll various types of virtual dice. It's useful for board games, TRPG (tabletop role-playing games), or making everyday decisions.

How to Use

  1. Select the number of dice to roll (1-6)
  2. Choose the dice type (D4, D6, D8, D10, D12, D20)
  3. Click the 'Roll Dice' button
  4. View the results with animation
  5. Check previous results in the roll history

Dice Type Descriptions

  • D4 (Tetrahedron): Pyramid-shaped die, values 1-4
  • D6 (Cube): Most common six-sided die, values 1-6
  • D8 (Octahedron): Eight-sided die, values 1-8
  • D10 (Pentagonal): Ten-sided die, values 1-10
  • D12 (Dodecahedron): Twelve-sided die, values 1-12
  • D20 (Icosahedron): Twenty-sided die, values 1-20 (most used in TRPG)
