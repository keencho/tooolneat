Screen Burn-In Test

Check for burn-in on OLED displays. If you see ghost images on gray screen, your display has burn-in.

How to Use

  • • Click pattern → Fullscreen mode
  • • ← → or click to change pattern
  • • ESC to exit

What is Burn-In?

Burn-in is permanent image retention on OLED displays when the same image is displayed for extended periods.

What is a Screen Burn-In Test?

A screen burn-in test is a diagnostic tool specifically designed to detect image retention and burn-in on OLED and plasma displays. By displaying uniform gray screens and test patterns, you can easily identify permanent ghost images that have been etched into your display from prolonged static content. This test is essential for evaluating used OLED TVs, monitors, and smartphones before purchase.

How to Use

  1. Click on a test pattern button (50% Gray is best for detecting burn-in)
  2. Enter fullscreen mode and look for any ghost images or uneven coloring
  3. Use arrow keys to cycle through different patterns (gray, gradient, checker, sliding)
  4. The sliding bar pattern helps identify burn-in that may be hidden in static images
  5. Press ESC to exit and document any issues found

Frequently Asked Questions

Q: What is screen burn-in and what causes it?

Burn-in is permanent image retention that occurs when static elements are displayed for extended periods on OLED or plasma screens. Common causes include TV channel logos, news tickers, game HUDs, taskbars, and navigation bars. The organic compounds in OLED pixels degrade unevenly when displaying the same image, causing permanent discoloration.

Q: What is the difference between burn-in and image retention?

Image retention is temporary and usually disappears after displaying different content or turning off the screen for a while. Burn-in is permanent damage that remains visible regardless of what content is displayed. Modern OLEDs include features to reduce both issues, but burn-in can still occur with static content over thousands of hours.

Q: Can screen burn-in be fixed?

Unlike stuck pixels, burn-in on OLED displays is permanent physical degradation and cannot be truly fixed. Some TVs have pixel refresh or screen shift features that can reduce visibility of minor burn-in, but significant damage is irreversible. Prevention through varied content, screen savers, and automatic brightness adjustment is the best approach.
